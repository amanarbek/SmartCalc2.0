Калькулятор

Бинарные операторы:

  • Плюс: +
  • Минус: -
  • Умножить: *
  • Делить: /
  • Возвести в стпень: ^
  • mod: остаток от деления числа

Унарные функции:

  • sin: синус угла
  • cos: косинус угла
  • tan: танегенс угла
  • asin: арксинус
  • acos: арккосинус
  • atan: арктангенс
  • sqrt: корень
  • ln: натуральный логарифм
  • log: десятичный логарифм
  • - :унарный минус
  • + :унарный плюс

Построение графиков:

PLOT: построение графика с заданными значениями областей
  • x min минимальное значение области определения
  • x max максимальное значение области определения
  • y min минимальное значение области определения
  • y max максимальное значение области определения

Дополнительные кнопки

  • = - вычисление введенного выражения
  • . - чисело с плавающей точкой
  • x - неизвестная переменная
  • backSpace - удаление последнего символа
  • C - удаление всего выражения
  • ( - открывающаяся скобка
  • ) - закрывающаяся скобка
